REST

endpoint - http://stickyapi.alanedwardes.com/1.0 // API version

POST /user/login
username=user&password=pass

Sucessful response:
{
	session: session object containing token ID,
	user: user object
}

Error codes: 403

POST/user/register
firstName, surname, password, email
success: 201
error: 500
invalid: 400

Get user object
/user/get/:id
Sucessful response:
{
	id:10000000001,
        firstName: 'Cool',
        surname: 'McRomance',
        email: 'test@testmail.com',
}

Error codes: 401
